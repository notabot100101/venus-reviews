# Venus Website Phase 2 Recovery Plan

## Overview
**Project:** Venus Website Recovery & Enhancement  
**Goal:** Transform Hugo site from basic structure to fully featured product review site  
**Timeline:** Phased execution with review gates  
**Team:** Architect (coordination), Forge (implementation), Pixel (images), Sophia (comms)

---

## Phase 1: Foundation (2-3 hours)

### Objectives
- Audit current site state (what's broken, what works)
- Fix any critical issues (broken links, missing assets)
- Create content structure for product reviews
- Establish clean baseline for Phase 2

### Tasks
1. **Site Audit**
   - Check Hugo site configuration and build status
   - Verify all product pages exist and link correctly
   - Identify missing assets (images, CSS, JS)
   - Test homepage and product pages for functionality
   - Document broken links and missing content

2. **Critical Fixes**
   - Fix broken links (if any found)
   - Ensure all product images are accessible
   - Verify Hugo build compiles without errors
   - Clean up any orphaned templates or layouts

3. **Content Structure**
   - Create review template for product pages
   - Set up rating system structure
   - Prepare affiliate link integration points
   - Establish content sections (features, specs, verdict)

### Deliverables
- ✅ Site audit report (screenshots + findings)
- ✅ Clean baseline site
- ✅ Content structure defined
- ✅ Fix confirmations

### Review Checkpoint
- Paul approves foundation completion
- Review items: audit report, baseline state, structure

---

## Phase 2: Content — Product Reviews (4-6 hours)

### Objectives
- Write 15 real product reviews (no placeholders)
- Include affiliate links, ratings, key features
- Update Hugo product pages with review content

### Tasks
1. **Content Creation**
   - Research each of 15 products
   - Write comprehensive review for each product
   - Include: specs, features, pros/cons, ratings
   - Add affiliate links in proper format
   - Write verdict/purchase recommendation

2. **Hugo Integration**
   - Create review template in Hugo layout
   - Add product-specific review content
   - Ensure proper markdown formatting
   - Test all reviews render correctly

3. **Quality Review**
   - Review sample products (3-5) with Paul
   - Adjust content based on feedback
   - Complete remaining reviews

### Deliverables
- ✅ 15 product reviews published
- ✅ Affiliate links integrated
- ✅ Rating system in place
- ✅ Sample reviews approved

### Review Checkpoint
- Paul reviews 3-5 sample products
- Approval to complete remaining reviews

---

## Phase 3: Visuals — Hero + Featured (3-4 hours)

### Objectives
- Generate new hero banner (no text, compelling visual)
- Create featured products grid for homepage
- Add trust badges, CTAs

### Tasks
1. **Hero Banner**
   - Generate compelling hero image (no text)
   - Ensure brand-appropriate visual
   - Optimize for homepage hero
   - Test rendering in Hugo

2. **Featured Grid**
   - Design featured products layout
   - Create product cards with images
   - Add hover effects and interactions
   - Integrate into homepage

3. **Trust Elements**
   - Add trust badges (security, satisfaction)
   - Add CTA buttons (Shop Now, Learn More)
   - Ensure responsive design
   - A/B test layouts if needed

### Deliverables
- ✅ Hero banner (screenshot + live URL)
- ✅ Featured products grid
- ✅ Trust badges & CTAs integrated
- ✅ Homepage transformed

### Review Checkpoint
- Paul approves homepage design
- Review items: hero banner, featured grid, homepage

## Execution Rules

1. **Sequential Phases** - Don't start Phase 2 until Phase 1 reviewed and approved
2. **Evidence Required** - Screenshots, live URLs for each deliverable
3. **Review Gates** - If phase fails review, fix before continuing
4. **No Parallel Work** - Prevent file conflicts (one phase at a time)
5. **Daily Updates** - Update Paul on progress each day
6. **Batch Processing** - Phase 4 processes 2-3 products per batch
